Структура проекта:

index.html
style.css
js/app.js
assets/good.gif
assets/bad.gif
assets/good.ogg
assets/bad.ogg

Важно:
1. Для финальных экранов положите свои файлы в папку assets с точными именами:
   good.gif, bad.gif, good.ogg, bad.ogg
2. Игра открывается простым двойным кликом по index.html.
3. Рекорды хранятся локально в браузере через localStorage.
4. На мобильном устройстве:
   - тап сверху/снизу/слева/справа от героя двигает его в эту сторону;
   - удержание продолжает движение до стены, вопроса или конца пути.
